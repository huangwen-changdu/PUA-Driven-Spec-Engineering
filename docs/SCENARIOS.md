# 📋 常见场景模板

> **目标**：提供常见场景的模板，减少重复描述，提升协作效率。

## 一、简单修改场景

### 场景 1：添加新字段
```
输入：在 [表名] 表添加 [字段名] 字段，类型 [类型]，用于 [用途]

预计步骤：2-3 步

```

### 场景 3：修改文档

输入：更新 [文档文件] 中的 [章节] 内容

## 十一、使用模板的注意事项

### 11.1 模板不是万能的
- 模板只是起点，AI 会根据实际情况调整
- 复杂需求可能需要多个模板组合
- 模板中的 `[占位符]` 需要替换为实际内容

### 11.2 模板的灵活性
- 你可以自由组合不同场景的模板
- 模板中的步骤数是估计值，实际可能更多或更少
- AI 会根据风险等级和成熟度调整流程

### 11.3 模板的更新
- 随着项目发展，模板可能需要更新
- 你可以提出新的场景模板
- 模板会根据用户反馈持续优化

---

> **记住**：模板的目的是减少重复描述，提升协作效率。AI 会根据你的需求自动判断使用哪个模板，但了解这些模板可以让你更清晰地表达需求。
